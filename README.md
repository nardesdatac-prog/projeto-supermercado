# 📊 Projeto de Análise de Dados - Supermercado

## 📌 Objetivo
Realizar análises estatísticas e criar visualizações interativas a partir de uma base de dados de produtos.

## 🔧 Tecnologias utilizadas
- Python
- Pandas
- Plotly

## 📊 Análises realizadas
- Média, mediana, variância e desvio padrão
- Análise por categoria
- Distribuição de preços
- Relação entre preços

## 📈 Visualizações
- Gráficos interativos com Plotly

## 🚀 Como executar
1. Clone o repositório
2. Abra o notebook no Jupyter
3. Execute as células
