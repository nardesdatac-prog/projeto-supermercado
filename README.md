# 🛒 Projeto de Análise de Dados - Supermercado

## 📌 Objetivo
Este projeto tem como objetivo analisar dados de produtos de supermercado, aplicando conceitos de estatística descritiva e criando visualizações interativas para gerar insights relevantes.

## 📊 Análises realizadas
- Média, mediana, variância e desvio padrão
- Distribuição de preços
- Análise por categoria
- Relação entre preço normal e desconto

## 📈 Visualizações
Foram desenvolvidos gráficos interativos utilizando Plotly para facilitar a exploração dos dados.

## 💡 Principais insights
- Algumas categorias concentram maior valor total de vendas
- A maioria dos produtos possui preços em faixas mais baixas
- Existe correlação entre preço original e preço com desconto

## 🛠️ Tecnologias
- Python
- Pandas
- Plotly

## 🚀 Como executar
1. Clone o repositório
2. Abra o notebook no Jupyter
3. Execute as células
